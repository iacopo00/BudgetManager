# BudgetManager

BudgetManager is a Java desktop application designed to help **basketball referees** manage and track their personal finances related to refereeing activities.

The project was developed for **academic purposes**, following the specifications provided by the course professor, and focuses on applying good software engineering practices using Java technologies.
